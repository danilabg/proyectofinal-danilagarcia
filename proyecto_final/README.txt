Hola!! Mi nombre es Danila García y este proyecto se trata de mi propio Portfolio, mi curriculum digital, y lo creé como una forma de mostrar lo que voy aprendiendo sobre Desarrollo Web. Obviamente a medida que incorpore nuevos lenguajes y conocimientos, iré aplicando procesos más complejos y completos. Lo bueno de esto, es que siempre se puede mejorar!!
En el proyecto utilicé una estructura HTML, le agregué estilos en cascada CSS y, además, algo del framework Bootstrap para darle una visual más profesional. También agregué Java Script, utilizando un evento para que los botones de la página cambien el color de claro a oscuro, no sólo el color de la tipografía sino también, el fondo.  

Es un proyecto que, si bien me costó, no dejé de disfrutarlo al ver que poco a poco fui logrando lo que quería.
Gracias por navegarlo!
